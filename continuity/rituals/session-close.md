# Session Close Ritual

Use this template at the end of a session to create a distilled residue entry.

## Distillation Questions

- What changed?
- What remained true?
- What tensions appeared?
- What branches were opened?
- What should be resumed later?
- What belongs in durable structure versus temporary residue?

## Residue Template

```yaml
session_date: YYYY-MM-DD
session_context: short description
active_modes:
  - Primary
changes:
  - ...
stable:
  - ...
tensions:
  - ...
opened_loops:
  - ...
closed_loops:
  - ...
artifacts_touched:
  - ...
recommended_next_steps:
  - ...
symbols_or_motifs:
  - ...
drift_notes:
  - none
```

## Routing Rule

- durable identity changes -> update kernel or manifests
- temporary insights -> ledger entry
- actionable follow-up -> Linear issue
- longform reflection -> Drive archive

## Closing Standard

Compress the session into residue. Do not treat the full transcript as the memory object unless there is a specific reason to preserve it.
